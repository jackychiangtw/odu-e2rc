Detailed Description

Structure to hold an Acknowledged Mode header.

  • dc : Data/Control PDU
  • rf : Resegmentation flag
  • p : Poll bit
  • fi : Framing Info
  • e : Extension bit
  • lsf : Last segment flat
  • sn : Sequence number
  • so : Segment offset
  • numLi : Number of length indicators in the following array (li)
  • li : Length indicators
